`gsutil` sample commands

- `gsutil cp`           -- copy
- `gsutil rm`           -- delete
- `gsutil mv`           -- move
- `gsutil ls`           -- list the directory hierarchy
- `gsutil mb`           -- make a bucket
- `gsutil rb`           -- remove a bucket
- `gsutil rsync`        -- emulation of Unix utility
- `gsutil acl`          -- access change list
